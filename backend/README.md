Backend skeleton for MoveUp.

Usage:
1. cd backend
2. npm install
3. Copy .env.example to .env and set DATABASE_URL
4. npm run test-db

- package.json (scripts to test DB)
- connect.js (simple DB connection test using knex)
- .env.example
- tsconfig.json
