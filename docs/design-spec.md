# 设计说明
## 配色方案
- 主色：#FFFFFF
- 辅助色：清新薄荷绿 #B4E6D9 或 #8FD6C5  柔和珊瑚橙： #FFC7B3 或 #FFB092
- 强调色：主强调色 -> 动感橙 #FF6B4A 或 #FF8A5C 次强调色-> 科技靛蓝 #4A7AFF 或 #5E9CEA     
- 中性色：雪白/近白 #F8FAFC #F2F5F9 浅灰-> #E9ECF0 #D0D7DE 中灰-> #94A3B8 #64748B 深灰/炭黑-> #1E293B #0F172A

## 字体
- 标题：Fugaz one
- 正文：Poppins、Outfit

## 核心页面
1. 首页 - 功能说明
- 导航页面
    1、介绍跑步的相关知识，普及半马、全马等知识。
    2、实时记录最近两次的活动记录
    3、还有开启跑步的运动按钮
    4、寻找附近的跑步社区
2. 详情页 
- 跑步信息介绍
    详细介绍跑步的相关信息和知识。
- 历史跑步记录
    详细记录每一次的跑步记录，日期、跑步距离等详细记录。
- 定制跑步计划
    定制一周内星期一到星期天每一天的具体计划。
    点击星期一项目卡，将会跳转到具体的定制计划页面
- 跑步爱好者社群
    会显示一些跑步爱好者社群。
    点击这些跑步爱好者社群，会显示一些其他跑友分享的跑步记录，同时可以组织日常晨跑、夜跑、长距离拉练、马拉松 / 路跑赛事、健康分享。
- 实时跑步记录
    GPS提供实时位置信息，同时会显示当前的天气状况和实时跑步距离、跑步时间和消耗卡路里的量。

3. 隐私相关
- 登录页面和注册页面
- 个人信息设置页面
    包括用户名、邮箱、电话号码、密码等敏感信息
## 交互说明

- GET STARTED PAGE 点击 join us 跳转到 JOIN US page，点击Log in 跳转到 LOG IN page 

- 导航栏组件
    Home、History、Plan、Club、Profile分别对应各自界面，点击可以跳转到各自界面。

- Home 页面跳转逻辑
    点击Running Information跳转到Running Information Page
    Find The Club Near You 点击See all 可以跳转到Club页面。
    Running start Interactions交互选项卡，点击start跳转到running start界面

- History 页面跳转逻辑
    点击跑步历史记录选项卡会跳转到关联的跑友爱好者社群活动，可以观察到自己参与到了哪些打卡活动

- Plan 页面跳转逻辑
    点击计划选项卡跳转到具体一周中某一天的计划。

- Club 页面跳转逻辑
    点击社群选项卡跳转到具体的跑步爱好者社群。

- Profile 页面跳转逻辑
    点击Edit Profile可以跳转到编辑页面。
- 下拉刷新
