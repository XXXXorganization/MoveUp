## 遇到的问题和解决
1.问题：运行时出现 from double to float 类型不兼容报错，海拔数据无法正常获取。
解决：在代码中添加 (float) 强制类型转换，统一数据类型，修复编译错误。
2.问题：按钮背景颜色设置不生效，黑色按钮显示异常。
解决：删除 android:background ，保留 android:backgroundTint 属性，即可。
3.问题：ConstraintLayout 报错提示找不到同级 ID。解决：调整布局层级，避免内部控件引用外部 ID，规范约束关系。
