##
遇到的问题和解决
问题：在使用 Figma 进行界面设计时，缺乏自定义复杂部件的经验，自主创建各类界面部件效率低、效果不佳，影响整体设计进度和视觉呈现效果。
主动探索 Figma 生态功能，发现平台内置的社区部件库与搜索功能，通过检索复用其他设计师已创建并共享的优质部件；同时筛选适配项目风格的部件进行微调适配，既解决了部件创建的难题，又提升了界面设计的精致度和美观性，保证了设计效率与视觉质量。

## 心得体会
本次设计实践中，深刻体会到 Figma 相较于墨刀的显著优势：一方面，Figma 免费版即可支持大文件、多页面的复杂原型设计，无需受墨刀 “大文件收费” 的限制，满足了项目全流程的设计需求；另一方面，Figma 丰富的社区资源、灵活的组件复用机制，大幅降低了设计门槛，提升了设计效率和成品质量。此外，也意识到高效利用工具生态（如共享部件库）是提升 UI 设计效率的关键，后续会继续深挖 Figma 的进阶功能，进一步提升设计能力和交付质量。
